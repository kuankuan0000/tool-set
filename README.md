# 个人项目集合

这个仓库包含了我的各种实用工具和项目集合。每个子目录都是一个独立的项目，可以单独使用。

## 项目列表

### 1. Fast Share
`/fast-share`

一个轻量级的命令行文件分享工具，用于在局域网内快速分享文件。

主要特性：
- 一键启动临时 HTTP 服务器分享单个文件
- 自动检测并显示局域网 IP 地址
- 支持自定义端口和自动关闭功能
- 跨平台支持（Windows/macOS/Linux）

技术栈：Python
